# Place Order Instructions

## PlaceLimitOrder

Places a density limit order.

### Accounts
| # | Account | Writable | Signer |
|---|---------|----------|--------|
| 0 | Market | Yes | No |
| 1 | Shard | Yes | No |
| 2 | User | Yes | No |
| 3 | User Token | Yes | No |
| 4 | Vault | Yes | No |
| 5 | Owner | No | Yes |
| 6 | Token Program | No | No |

### Parameters
```rust
struct PlaceLimitOrderParams {
    price: u64,
    amount: u64,
    side: u8,
    expires_at: i64,
}
```

## PlaceClmmPosition

Places a concentrated liquidity position.

struct PlaceClmmPositionParams {
