# Rust SDK Installation

## Add Dependency

Add to your `Cargo.toml`:

```toml
[dependencies]
braid-client = { version = "0.1", features = ["fetch"] }
solana-client = "2.2"
solana-pubkey = "2.2"
```

<Note>
The `fetch` feature enables RPC-based fetching. Without it, only PDA derivation and types are available.
</Note>

## Basic Setup

```rust
use braid_client::{Market, BRAID_ID};
use solana_client::rpc_client::RpcClient;
use solana_pubkey::Pubkey;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let rpc = RpcClient::new("https://api.mainnet-beta.solana.com");

    // Load market
    let market_address: Pubkey = "...".parse()?;
    let market = Market::new(&rpc, market_address);
    let account = market.fetch()?;

    println!("Base mint: {}", account.data.mint_a);
    println!("Quote mint: {}", account.data.mint_b);

    Ok(())
}
```

## Derive Addresses

```rust
use braid_client::{
    generated::accounts::Market as MarketData,
    ClmmOrdersPda, LimitOrdersPda, BRAID_ID,
};

// Derive market PDA (static method on MarketData)
let market_pda = MarketData::derive_address(
    &mint_a,
    &mint_b,
    &[0u8; 8],  // 8-byte seed
    &BRAID_ID,
);

// Derive shard PDAs
let clmm_shard = ClmmOrdersPda::derive_address(&market_pda, shard_idx, &BRAID_ID);
let limit_shard = LimitOrdersPda::derive_address(&market_pda, shard_idx, &BRAID_ID);
```
