# TypeScript SDK Installation

## Requirements

- Node.js 18+ or Bun
- TypeScript 5.0+ (recommended)

## Install

<Tabs>
  <Tab title="npm">
    ```bash
    npm install @braid/client
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add @braid/client
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add @braid/client
    ```
  </Tab>
  <Tab title="bun">
    ```bash
    bun add @braid/client
    ```
  </Tab>
</Tabs>

## Peer Dependencies

The SDK requires `@solana/web3.js` v2 (which includes `@solana/kit`):

```bash
npm install @solana/web3.js@2
```

## Basic Setup

```typescript
import { createSolanaRpc, address } from '@solana/kit';
import { Market, BRAID_PROGRAM_ADDRESS } from '@braid/client';

// Connect to Solana
const rpc = createSolanaRpc('https://api.mainnet-beta.solana.com');

// Create market instance
const marketAddress = address('...');
const market = new Market(rpc, marketAddress);

// Fetch market data
const state = await market.fetch();

console.log('Market loaded:', market.address);
console.log('Base mint:', state.data.mintA);
console.log('Quote mint:', state.data.mintB);
```

## Environment Variables

For development, you may want to configure:

```bash
# .env
SOLANA_RPC_URL=https://api.mainnet-beta.solana.com
BRAID_PROGRAM_ID=BRAiD1111111111111111111111111111111111111
```
